Jayground

Данное приложение осуществляет работу с сущностями типа "квартира", имеющими следующую структуру:

id : long : стандартный уникальный идентификатор
floor : int : этаж
space : int : площадь
description : text : описание 
location : object
- latitude : double : широта
- longitude : double : долгота
